Which browsers work with Qlear

To have the best experience with Qlear on your computer, use a supported, up-to-date browser, you’ll have access to the latest features. For the best security and functionality, keep your web browser up to date.

If you’re using these browsers and need to upgrade to the latest version, go to the website for your browser:

  • Microsoft Edge

  • Internet Explorer

  • Chrome

  • Firefox

  • Safari
